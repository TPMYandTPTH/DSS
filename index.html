DDS

Document Divider System

Split your Word documents into 2-page job descriptions

Backend Server Required

This application requires a Python Flask backend to function.

To use this application:

  1. Install Python 3.8 or higher
  2. Save the backend code as app.py
  3. Install dependencies: pip install Flask python-docx flask-cors
  4. Run the server: python app.py
  5. Refresh this page
Quick Start:
pip install Flask python-docx flask-cors
python app.py
